# Program Pengelolaan Data Nilai Mahasiswa (Python Dasar)

**Tugas Praktikum Pemrograman Web - Pertemuan 1**

---

## I. Deskripsi Aplikasi

Aplikasi ini adalah program konsol sederhana yang dikembangkan menggunakan **Python 3** untuk mengelola dan memproses data nilai akademik mahasiswa. Program ini dirancang sebagai implementasi dasar dari konsep-konsep fundamental Python, termasuk penggunaan **List**, **Dictionary**, **Fungsi**, dan **Struktur Kendali** (`if/loop`).

## II. Fungsionalitas Utama

Program menyediakan menu interaktif dengan fungsionalitas utama sebagai berikut:

* **Perhitungan Nilai Akhir (NA):** Menghitung nilai akhir dengan bobot 30% UTS, 40% UAS, dan 30% Tugas.
* **Penentuan Grade Otomatis:** Menentukan grade (A hingga E) berdasarkan nilai akhir.
* **Tampilan Data:** Menampilkan seluruh data nilai dalam format tabel.
* **Input Data:** Fitur untuk menambahkan data mahasiswa baru.
* **Analisis Kelas:** Mencari nilai tertinggi, terendah, dan menghitung rata-rata kelas.
* **Filter Data:** Menyaring mahasiswa berdasarkan grade yang dicapai.

## III. Panduan Penggunaan

### A. Persyaratan Sistem

Program ini memerlukan **Python 3.8+** terinstal pada sistem Anda. Tidak ada modul eksternal (third-party library) yang dibutuhkan.

### B. Cara Menjalankan Program

1.  **Akses Direktori:** Buka terminal dan navigasikan ke folder tugas ini.
    ```bash
    cd [path_ke_folder_tugas]
    ```
2.  **Eksekusi Skrip:** Jalankan file program utama:
    ```bash
    python data_nilai_mahasiswa.py
    ```
3.  Ikuti petunjuk menu interaktif yang tersedia di terminal.
